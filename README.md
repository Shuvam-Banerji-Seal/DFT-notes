# DFT Notes

> Reader-first notes on Density Functional Theory, served as a static
> site on GitHub Pages.

This repository hosts a growing set of notes on Density Functional Theory
(DFT) — the standard workhorse of modern quantum chemistry and
computational materials science. The notes are written in Markdown with
LaTeX-style math and rendered as a static site via Jekyll on GitHub
Pages.

## Quick start

```bash
# Install Jekyll + plugins
bundle install

# Serve locally with hot-reload
bundle exec jekyll serve
# Open http://localhost:4000/
```

The site is then served at `http://localhost:4000/` (or, if you're using
the GitHub Pages baseurl, `http://localhost:4000/DFT-notes/`).
