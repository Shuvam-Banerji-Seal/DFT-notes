### Added

- Initial public release of the repository.
- Jekyll + KaTeX static site with full SEO meta tags, sitemap, and JSON-LD.
- GitHub Actions workflows: build + deploy to Pages, Markdown lint, dependabot.
- Issue templates (bug, content, feature) and pull request template.
- Notes:
  - `notes/01-many-body-qm.md` — many-body QM primer, BO approximation,
    Hartree–Fock, density definition.
  - `notes/02-hohenberg-kohn.md` — HK I & II, constrained-search formulation.
  - `notes/03-kohn-sham.md` — KS equations, SCF algorithm, what works and
    what doesn't.
  - `notes/04-exchange-correlation.md` — Jacob's ladder, LDA → GGA →
    meta-GGA → hybrid, practical guidance.
  - `notes/99-references.md` — foundational papers, textbooks, reviews,
    online resources.
- Comprehensive README with TOC, badges, SEO keywords, and module roadmap.
