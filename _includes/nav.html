{%- comment -%} Top navigation — sticky cream (or warm-dark in dark mode) canvas, 64px tall on desktop, auto on mobile (allows wordmark + actions to breathe). Renders the spike-mark + wordmark on the left, the primary menu in the middle, and a cluster of action items on the right: theme switcher, GitHub text-link, coral "Read the notes" CTA, and a hamburger button (mobile only). Adapted from the design.md spec: - `top-nav` background = canvas - menu type = nav-link (Inter 14px / 500) - primary CTA = button-primary (coral) - wordmark prefix = spike-mark (4-spoke radial) - theme switcher = small icon button, sun/moon swap on `[data-theme="dark"]` on Mobile menu (data-mobile-menu) is a right-side slide-in drawer with: - Backdrop overlay (click-to-dismiss) - Body scroll lock while open - Theme switcher in the drawer (since the top-nav theme toggle is hidden on mobile) - Active page indicator - Chapter quick-jump (00-17) at the top - Close button (X) at top-right - "Read the notes" CTA at the bottom - 44px minimum tap targets throughout State management: - data-mobile-open="true" on body when menu is open - aria-expanded="true" on hamburger when open - data-nav-link: any link inside the drawer (auto-closes on tap) {%- endcomment -%}
