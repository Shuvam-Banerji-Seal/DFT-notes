{#- Theme picker — design.md v2.0 -------------------------------------------------------------------------- 6 theme swatches (one per palette) + 3 mode buttons (light/dark/auto). All behavior is in assets/js/theme-picker.js (localStorage + OS-pref tracking + a custom 'dft-theme-changed' event so other code can react). The swatches use the *light* variant's canvas+accent as the preview. This is a deliberate trade-off — 6 swatches is plenty to recognise a theme at a glance without 12 dots. The picker is fixed top-right; on very narrow screens it stays small. -#}
